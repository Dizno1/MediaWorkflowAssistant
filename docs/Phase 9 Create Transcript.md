# Phase 9 - Create Transcript

Phase 9 converts Create Transcript from a starter worksheet into a complete production workflow.

## Workflow

1. Select a supported local audio or video file.
2. Open Create Transcript.
3. Use the existing media viewer and accessible transcript editor.
4. Enter spoken content, speaker labels when useful, and meaningful sounds.
5. Confirm the transcript was reviewed for accuracy and completeness.
6. Save through the existing execution engine.
7. Register the reviewed plain-text transcript with the Output Manager.
8. Record completion, word count, review time, history, and artifact metadata in Shared Knowledge.
9. Mark transcription Completed in recommendations.

## Accessibility

The editor uses a labeled native textarea, a native confirmation checkbox, status announcements, keyboard-operable controls, focus management, and the existing execution progress interface.

## Next Phase

Phase 10 will implement a production caption workflow with an accessible timed-cue editor and valid WebVTT output. It will reuse a completed transcript when available.
