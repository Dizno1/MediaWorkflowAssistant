# Phase 10 - Create Captions

Phase 10 converts Create Captions from a starter template into a complete production workflow.

## Workflow

1. Choose a local video containing audio.
2. Open Create Captions.
3. Reuse a completed transcript when available to create starter cues.
4. Review the video in the existing Viewer.
5. Edit each cue using labeled start time, end time, and caption text fields.
6. Add or remove cues as needed.
7. Validate timestamp format, chronological order, overlap, source duration, and caption text.
8. Confirm the captions were reviewed for accuracy, timing, reading order, speaker identification, and meaningful sounds.
9. Execute through the Workflow Execution Engine.
10. Export reviewed WebVTT captions and a caption review record.
11. Register both artifacts with the Output Manager.
12. Record cue count, review time, completion, and confidence in Shared Knowledge.
13. Mark Create Captions completed in recommendations.

## Accessibility

The editor uses native form controls, fieldsets and legends for each cue, explicit labels, keyboard-operable add and remove controls, a polite status region, error focus management, and section focus when the workflow opens.
