Sprint 5 Part 2

Viewer

Added a Viewer that appears after a file is selected.

Supported in this sprint

- Video playback
- Audio playback
- Image display
- Text and Markdown reading
- PDF viewing where supported by the browser
- Clear fallback message for unsupported file types

Accessibility

- Native media controls remain keyboard accessible.
- Video and audio controls include accessible labels.
- Text previews are keyboard focusable.
- PDF previews include a descriptive title.
- The Viewer does not replace the Current file summary or the goal-driven choices.
