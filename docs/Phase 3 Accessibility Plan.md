# Phase 3 Accessibility Plan

Phase 3 adds an ordered accessibility plan beneath the assessment.

The plan is generated from the shared media knowledge model and the accessibility assessment. It does not create a second analysis path.

For audio and video sources, the plan can include:

1. Analyze the speech.
2. Create a transcript.
3. Identify speakers.
4. Create captions.
5. Analyze visual content.
6. Recommend an audio description approach.
7. Create audio description.
8. Prepare one accessibility package.

Each step records:

- purpose
- order
- dependencies
- later work it supports
- current availability
- whether it is complete, available, preparation-only, or planned

The plan is passed into workflow jobs so later phases can update shared knowledge and avoid repeating completed work.

The future primary action is "Make this media accessible." It will use the plan to run only the work that is needed and available.
